# Start the dashboard (frontend)

Node.js and npm must be installed and available in your PATH.

1. Open a terminal (Command Prompt, PowerShell, or VS Code terminal).
2. Run:

```bash
cd ndtbundle-dashboard
npm install
npm run dev
```

3. Open **http://localhost:3000** in your browser.

The dashboard calls the API at **http://localhost:5000** by default. Ensure NdtBundleService is running on that port.
